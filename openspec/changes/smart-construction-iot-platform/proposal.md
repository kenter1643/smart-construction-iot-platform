## 为什么

建筑行业在安全、效率和实时监控方面面临重大挑战。传统方法劳动密集，缺乏及时的数据驱动洞察。智慧工地物联网平台可以通过实现对设备、机械和环境条件的实时监控来解决这些问题，从而提高安全性、降低成本并提高生产力。

## 变更内容

- **设备通讯**：添加对 Modbus TCP、MQTT 和 HTTP 协议的支持
- **视频流**：实现视频流接入、转发、存储和查看功能
- **视频查看**：支持实时视频流和录制视频的播放，包含播放控制、质量调整和全屏观看功能
- **实时监控**：提供设备和传感器数据的实时可视化仪表盘，支持视频查看组件
- **设备管理**：允许物联网设备的注册、配置和控制
- **数据存储与分析**：存储历史数据并提供分析功能
- **告警与通知**：针对异常情况（如高温、设备故障）发送告警
- **安全**：实现身份认证、授权和数据加密
- **可扩展性**：设计用于处理大量设备和高数据吞吐量
- **技术栈升级**：前端采用 Vue 3 + 通用组件库（Element Plus），后端核心服务采用 Rust（Axum），关系数据库采用 MySQL，并保留 InfluxDB 处理时序数据

## 影响

- **后端服务**：新增设备通讯、数据管理和视频流的 API 端点，并逐步迁移核心路径到 Rust 服务
- **前端**：新增实时监控和设备管理的仪表盘及 UI 组件，采用 Vue 组件化实现
- **数据库**：采用 MySQL 存储关系数据，InfluxDB 存储时序数据和指标
- **基础设施**：需要边缘计算支持、高带宽连接和可靠的数据处理
