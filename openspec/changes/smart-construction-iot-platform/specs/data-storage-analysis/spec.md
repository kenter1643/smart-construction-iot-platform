## 新增需求

### 需求：历史数据存储
系统应存储历史传感器和设备数据。

#### 场景：存储传入的传感器数据
- **当** 平台从设备接收传感器数据时
- **则** 系统应存储带有时间戳、设备 ID 和传感器值的数据

### 需求：数据查询
系统应允许用户查询历史数据。

#### 场景：按时间范围查询数据
- **当** 用户提供开始时间和结束时间时
- **则** 系统应返回该时间范围内的所有数据记录

### 需求：数据分析
系统应提供基本的数据分析功能。

#### 场景：计算平均传感器值
- **当** 用户请求特定传感器在时间范围内的平均值时
- **则** 系统应计算并返回平均值
