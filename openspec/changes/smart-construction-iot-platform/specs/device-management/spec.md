## 新增需求

### 需求：设备注册
系统应允许用户注册新设备。

#### 场景：使用唯一 ID 注册新设备
- **当** 用户提供唯一的设备 ID 和设备类型时
- **则** 系统应创建新的设备记录并将其存储在数据库中

### 需求：设备配置
系统应允许用户配置已注册的设备。

#### 场景：更新设备配置
- **当** 用户修改设备的配置参数（如通讯协议、采样率）时
- **则** 系统应保存新配置并将其应用到设备

### 需求：设备控制
系统应允许用户向已注册的设备发送控制命令。

#### 场景：向设备发送控制命令
- **当** 用户向已注册的设备发送有效的控制命令时
- **则** 设备应执行命令并返回指示成功或失败的响应
